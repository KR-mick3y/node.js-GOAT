Document Viewer - User Guide
============================

Getting Started
---------------
1. Navigate to the main page to see available documents.
2. Click "View" next to any document to open it.
3. Documents are served in plain text format.

File Access
-----------
Documents are stored in the /app/files/ directory.
Only files within this directory are intended to be accessible.

Security Notice
---------------
If you discover any security issues with this system,
please report them to the security team immediately.
